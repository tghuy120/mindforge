# Agent Search Tools

## 摘要

Exa、Tavily 和 Context7 是 AI Agent 的三大搜索工具，覆盖不同搜索需求象限：Exa 用于深度语义研究，Tavily 用于实时事实搜索，Context7 用于版本特定技术文档。传统搜索引擎返回"链接列表"不适合 Agent 消费，Agent 需要结构化、可直接消费的内容。
